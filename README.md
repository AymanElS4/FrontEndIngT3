# FrontEndIng

## Requisitos

- Node.js 18 o superior
- npm

## Instalación

1. Abrir una terminal en la carpeta del frontend:

```bash
cd "c:\Users\Ayman El Salous Mnz\OneDrive\Desktop\FrontEndIng\FrontEndIng"
```

2. Instalar dependencias:

```bash
npm install --legacy-peer-deps
```

## Ejecución

Para iniciar el frontend en modo desarrollo:

```bash
npm run dev -- --host
```

Esto levantará el servidor de Vite en:

- http://localhost:5173/

## Construcción

Para generar los archivos de producción:

```bash
npm run build
```

## Lint

Revisar el código con ESLint:

```bash
npm run lint
```

Corregir problemas automáticamente:

```bash
npm run lint:fix
```
